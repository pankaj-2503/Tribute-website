# Tribute Website

This is a tribute website I built using HTML and CSS. It is a simple website that celebrates the life and accomplishments of a loved one, friend, or mentor. The website includes a biography, photos, and videos, as well as a section for comments and tributes from others.

## How to Use

To use this website, simply navigate to the URL in your web browser. The website will automatically load and you can begin exploring.

## Features

This website includes the following features:

* A biography of the honoree
* Photos and videos of the honoree
* A section for comments and tributes from others
* A responsive design that looks great on all devices

## Technologies Used

This website was built using the following technologies:

* HTML
* CSS
